⚖️ Менялы vs Автомат

🏪 Менялы (Биржа / CEX)

Кто устанавливает цену?
Люди. Каждый меняла ставит свой курс.
Кто зарабатывает?
Менялы — на спреде между покупкой и продажей.
Кто может быть менялой?
Только крупный игрок. На Binance маркет-мейкер = фирма с миллионами.
Что если менял нет?
Нет ликвидности → нет обмена. Стоишь и ждёшь.
Прозрачность
Видишь книгу ордеров, но не знаешь кто за ними стоит.

🤖 Автомат (Пул / AMM)

Кто устанавливает цену?
Никто. Формула x × y = k рассчитывает курс автоматически.
Кто зарабатывает?
Все, кто положил деньги в автомат. Комиссия 0.05–1% с каждого обмена.
Кто может быть «менялой»?
Любой. Положил $10 в пул — ты уже маркет-мейкер.
Что если никого нет?
Автомат работает 24/7. Пока есть токены внутри — обмен возможен.
Прозрачность
Всё на блокчейне. Видно каждый токен, каждую комиссию.
💡 Главная мысль: DeFi убрал посредника. Раньше менять деньги мог только крупный игрок. Теперь это может любой — через формулу.
