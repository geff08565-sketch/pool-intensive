📐 Формулы V3 (для тех кто хочет разобраться):
L (ликвидность) = рассчитывается из внесённых токенов и диапазона
ETH в позиции = L × (1/√P − 1/√Pb) — когда цена в диапазоне
USDC в позиции = L × (√P − √Pa) — когда цена в диапазоне
Цена ≥ верхней → ETH = 0, USDC = L × (√Pb − √Pa)
Цена ≤ нижней → ETH = L × (1/√Pa − 1/√Pb), USDC = 0

🎯 Uniswap V3

Диапазон
Ты выбираешь сам: узкий или широкий.
Эффективность
Высокая. Все деньги работают в твоём коридоре.
Управление
Нужно следить. 5-10 мин в день.
Риск
Выше — цена может выйти за диапазон.
Пропорция
Зависит от диапазона. Может быть 70/30, 80/20.
Для кого
Активное управление. Больше дохода = больше внимания.

🔑 Широкий vs Узкий диапазон

🟢 Широкий ($1,500–$4,000)
• Меньше комиссий на единицу ликвидности
• Реже выходишь из диапазона
• Спокойнее, меньше стресса
• Лучше для начала
🟠 Узкий ($2,400–$2,600)
• Больше комиссий (концентрация!)
• Часто выходишь — нужно ребалансить
• Для опытных, кто следит за рынком
• Выше доход = выше риск
💡 Золотое правило для новичков: ±20-30% от текущей цены. Не слишком узко, не слишком широко. С опытом сузишь.
